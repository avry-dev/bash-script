# bash-script
A collection of sysadmin scripts written while learning Linux and bash.

## Scripts
### syshealth.sh
Displays general system health information like hostname, uptime, CPU usage and temperatures, RAM and Swap usage, and disk usage.
Usage: bash syshealth.sh

### disk-alert.sh
Checks root partition disk usage and warns if it exceeds 80%.
Usage: bash disk-alert.sh

### service-checker.sh
Checks the status of common services (ssh, cron, nginx) and prints whether each is active or inactive.
Usage: bash service-checker.sh

## What i learned
Writing these taught me variables, conditionals, loops, command substitution, and log parsing in bash.
